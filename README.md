Materials Needed

1)Arduino Uno or Arduino Nano
2)Motor Driver Module
3)Line Sensor Module (IR sensor)
4)Battery Holder (included with the kit)
5)4 1.5V batteries
6)Jumper Wires
7)Chassis frame
8)Wheels and motor mounts
9)Front wheel (included in the kit)
10)Screws, nuts
11)Chassis brackets
12)double sided tape

Optional
1)breadboard for connections
2)welding machine for wires
